# Online Quiz Management System

## Overview

The Online Quiz Management System is a web-based application developed using Java Servlets, JDBC, MySQL, HTML, and CSS. The system allows users to register, log in, attempt quizzes across multiple subjects, and view detailed results with score history.

## Features

* User Registration and Login Authentication
* Quiz Modules for:

  * Java
  * Database Management System (DBMS)
  * Data Structures and Algorithms (DSA)
* Unlimited Quiz Attempts
* Automatic Score Calculation
* Detailed Result Analysis

  * Correct Answers
  * Incorrect Answers
  * Answer Review
* Quiz Attempt History
* Secure Database Storage using MySQL
* Responsive User Interface

## System Workflow

1. User registers and logs into the system.
2. User selects a quiz category (Java, DBMS, or DSA).
3. User attempts the quiz.
4. System evaluates answers and calculates the score.
5. Results are displayed with correct and incorrect answers.
6. Quiz history and scores are stored in MySQL.
7. Users can attempt quizzes multiple times and track their progress.
